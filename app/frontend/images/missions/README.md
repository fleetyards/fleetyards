# Mission Cover Images

Default cover images for the mission builder, keyed by `Mission.category`.

Drop in WebP files (recommended ~1600×500 for the banner; smaller is fine):

| Category | File |
| --- | --- |
| `ship_combat` | `ship_combat.webp` |
| `ground_combat` | `ground_combat.webp` |
| `combined_combat` | `combined_combat.webp` |
| `mining` | `mining.webp` |
| `salvage` | `salvage.webp` |
| `cargo_hauling` | `cargo_hauling.webp` |
| `exploration` | `exploration.webp` |
| `other` | `other.webp` |

When a file is missing, the UI falls back to the generic placeholder.
