# Fleetyards.net

A Ship Database and Web API based on the official [Star Citizen Ship Matrix](https://robertsspaceindustries.com/ship-specs).

Powered by [Ruby on Rails 6.x](https://rubyonrails.org/) and [VueJS 2.x](https://vuejs.org/)
